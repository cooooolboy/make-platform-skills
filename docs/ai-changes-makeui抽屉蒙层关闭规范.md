# makeui 抽屉蒙层关闭规范

## 背景

用户明确要求新建、编辑和详情抽屉默认支持点击蒙层关闭。

## 变更内容

- 更新 `skills/makeui/SKILL.md`，将版本提升到 `0.3.12`，新增 create/edit/detail Drawer 默认可点击蒙层关闭的规则。
- 更新 `skills/makeui/references/drawer-layout.md`，规定 Drawer 默认 `maskClosable`，点击蒙层或空白区域关闭当前抽屉。
- 规定蒙层关闭应走与头部关闭按钮相同的关闭路径；如果存在未保存变更保护，则同样应用到蒙层关闭。
- 规定堆叠抽屉中点击蒙层只能关闭最上层抽屉；如果不能保证这一点，应修复抽屉栈关闭逻辑，而不是禁用蒙层关闭。

## 影响范围

- 后续生成的新建、编辑和详情 Drawer 默认支持点击蒙层关闭。
- 堆叠 Drawer 的关闭行为会更一致，不再通过禁用蒙层关闭来规避栈管理问题。
